## 介绍

一个开箱即用的大前端脚手架创建工具。

## 安装

```bash
npm i shzl-cli -g
```

## wy init

初始化一个工程。支持在已经创建好的目录下执行这个命令，例如在 `project-name` 工程下执行 `shzl-cli`,就会出现? 请输入项目名称 输入项目名即可。

目前支持的脚手架模板：

```bash
◯ miniapp
◯ pc
◯ mobile
◯ lib-rollup
◯ antvue
◯ dd
◯ wechat
```
